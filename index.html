跳跃游戏

解题思路

设n为你可以跳跃到的最远的位置，从0开始。

采用循环，保证从前往后每一个位置都能够被判断。

i+nums[i]表示的是，如果在第i个位置的话，可以到达的最远位置

在某点可以到达的最远的位置大于你当前可以跳跃的最远的位置，需使得n一直保持在最远的位置的值，那么就要令n=i+nums[i]。

估计了n值，即可知道每个位置所能够到达的长度（位置），如果没有一个位置的长度能到达终点，则一定不能跳跃到最远的位置 false

如果n大于等于最后一个元素的下标（numsSize-1），即一定能到达最远位置 true

当数组的下标小于等于1时，一定能到达最远的位置 true

当数组中有一个下标所代表的数为0，且你只能跳到这个位置上时，一定不能到达最远的位置 false

盛最多水的容器

解题思路

采用双指针的方法，

由于结果由两个指针指向的数字中较小值∗指针之间的距离所决定，

所以两个指针分别位列左右两头，将两条边与其左/右侧的高度做对比，移动数值较小的边，如果移动数值较大的边，那么能装的水一定会变少，而移动数值较小的一边则不一定，每次移动记录所得到的结果。
