Nim游戏

解题思路

首先需注意，“我”作为游戏的先手且我们每一步都是最优解。

接下来依次列举可能性：

n=1时，我拿走一块石头 false

n=2时，我拿走两块石头 false

n=3时，我拿走三块石头 false

n=4时，不论我拿走一块/两块/三块石头，朋友都可以选择拿走三块/两块/一块石头 true

n=5时，我先拿走一块石头，接下来不管朋友拿走几块，都是我赢 false

n=6时，我先拿走两块石头，接下来不管朋友拿走几块，都是我赢 false

n=7时，我先拿走三块石头，接下来不管朋友拿走几块，都是我赢 false

n=8时，不论我拿走几块石头，只要第一轮过去，场上的石头还有四个，就是朋友赢 true

n=9时，……
